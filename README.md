## 🤝 Como contribuir

Quer colaborar com este repositório? Toda ajuda é bem-vinda! Aqui vão algumas formas de contribuir:

- Sugira novos links: Achou uma ferramenta ou site incrível? Abra uma issue ou envie um PR com uma breve descrição e o link.
- Melhore descrições: Viu algo que pode ser explicado melhor? Fique à vontade para editar e deixar mais claro.
- Organize categorias: Se notar que algo pode ser melhor agrupado ou renomeado, abra uma discussão.
- Mantenha o padrão: Sempre verifique se o recurso é gratuito (ou tem opção gratuita) e mantenha o estilo da tabela.

### Passo a passo

#### 1. Faça um fork do repositório

#### 2. Clone seu fork
```git clone https://github.com/seu-usuario/frontend-de-bolso.git```

#### 3. Crie uma nova branch
```git checkout -b minha-sugestao```

#### 4. Faça suas alterações e commit
```git commit -m "Adiciona ferramenta X na seção Y"```

#### 5. Envie para seu fork
```git push origin minha-sugestao```

#### 6. Abra um Pull Request no repositório original
